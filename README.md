# Splunk Detection Rules

A collection of Splunk SPL detection rules focused on Windows Security monitoring, authentication events, account management, and SOC detection engineering.

## Repository Structure

```
splunk-detection-rules/
│
├── authentication/
├── account-management/
└── references/
```

- After Hours Login Detection *(Coming Soon)*

- Splunk Enterprise
- SPL (Search Processing Language)
- Windows Security Event Logs
- Windows Event IDs
- MITRE ATT&CK Framework

## Disclaimer

All detection rules are based on lab environments and educational use cases. No production or company data, logs, IP addresses, usernames, or confidential information are included.
